# 📧 Spam Email Classifier (Web App)

This is a Machine Learning web app built using Streamlit that classifies messages as Spam or Not Spam.

## 🚀 Features
- User-friendly web interface
- Real-time spam detection
- Built using Machine Learning

## 🛠️ Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas
